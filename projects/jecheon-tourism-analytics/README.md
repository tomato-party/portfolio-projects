# Jecheon Tourism Analytics

## Summary

제천시 관광, 맛집, 문화행사 데이터를 활용해 지역 활성화 관점의 인사이트를 도출한 공모전 프로젝트입니다.

## Problem

지역 관광 데이터와 문화 콘텐츠 데이터를 결합해 실행 가능한 정책/서비스 아이디어를 발굴합니다.

## Approach

공공데이터 수집, 관광지/음식점/입장객 데이터 분석, 음악영화제 장르 크롤링 아이디어

## Tech Stack

Python, pandas, public data analysis
