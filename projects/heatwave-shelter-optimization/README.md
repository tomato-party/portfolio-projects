# Heatwave Shelter Network Optimization

## Summary

폭염 취약지역을 대상으로 기존 쉼터 네트워크를 재설계하고, 입지 최적화 관점에서 더 효율적인 쉼터 배치 전략을 제안한 산업공학/공공데이터 분석 프로젝트입니다.

## Problem

폭염 위험이 커지는 상황에서 취약계층이 실제로 접근 가능한 쉼터 네트워크를 설계하는 것이 중요합니다. 이 프로젝트는 쉼터의 단순 개수보다 접근성, 커버리지, 취약지역 우선순위를 함께 고려해 입지 재설계 문제를 정의했습니다.

## Approach

- 폭염 취약지역과 기존 쉼터 네트워크 구조 분석
- 수요지점과 후보지의 공간적 관계를 고려한 입지 최적화 문제 설계
- 취약지역 커버리지 개선을 목표로 한 쉼터 네트워크 재배치/확충 방향 제안
- 정책 제안형 보고서로 문제정의, 분석 흐름, 기대효과 정리

## Tech Stack

Public data analysis, location optimization, network redesign, spatial decision modeling
