# AI Trading Competition

## Summary

AI 트레이딩 대회를 위한 금융 데이터 기반 예측/전략 프로젝트입니다. 보고서 중심으로 투자 아이디어, 모델링 접근, 성과 해석을 정리했습니다.

## Problem

시장 데이터를 활용해 예측 가능한 신호를 찾고, 이를 거래 의사결정으로 연결하는 전략을 설계합니다.

## Approach

금융 데이터 전처리, 피처 설계, 모델 기반 예측, 백테스트/성과 해석 관점의 보고서 구성

## Tech Stack
