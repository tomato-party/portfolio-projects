# House Price Prediction

## Summary

Kaggle house price prediction 스타일의 회귀 모델링 프로젝트입니다.

## Problem

주택 특성 데이터를 바탕으로 SalePrice를 예측합니다.

## Approach

범주형/수치형 결측치 처리, 이상치 처리, 피처 엔지니어링, CatBoost/LightGBM/XGBoost ensemble

## Tech Stack

Python, pandas, scikit-learn, CatBoost, LightGBM, XGBoost
